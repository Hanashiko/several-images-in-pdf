# Images to PDF

A simple Python script that combines multiple images into a single PDF file using [img2pdf](https://pypi.org/project/img2pdf/).

## Usage

1. Set the `path` variable to the directory containing your images.
2. Run the script:

```bash
python main.py
```

The output file `result.pdf` will be created in the current directory.

## Requirements

- Python 3
- img2pdf (`pip install img2pdf`)
